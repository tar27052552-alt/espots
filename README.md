# ระบบรายงานผลการสมัครแข่ง Free Fire Tournament Dashboard 🎮🔥

ระบบแสดงข้อมูลผู้สมัครการแข่งขันเกม Free Fire ในรูปแบบหน้าเว็บ Dashboard สำหรับบอร์ดควบคุมระบบสถิติ ค้นหา และกรองข้อมูลได้อย่างรวดเร็ว พัฒนาขึ้นโดยใช้ **Google Apps Script Web App** เพื่อเชื่อมข้อมูลโดยตรงกับ Google Sheets แบบ Real-time!

---

## 📁 ไฟล์ระบบที่มีให้
1. **`Code.gs`** : โค้ดส่วนหลังบ้าน (Backend) ทำหน้าที่ดึงข้อมูลจากชีตและส่งไปให้ส่วนหน้าจอแสดงผล
2. **`Index.html`** : โค้ดส่วนหน้าจอ (Frontend Dashboard) ตกแต่งด้วย CSS สไตล์ Esports/Gaming ธีมสีมืด (Neon Cyan & Red) มีฟังก์ชันคำนวณสถิติ ค้นหารายชื่อ กรองตามระดับชั้น และดูแยกแบบทีมหรือแบบรายบุคคล

---

## 🚀 ขั้นตอนการติดตั้งใช้งาน (Deployment)

ทำตามขั้นตอนด้านล่างนี้เพื่อเปิดใช้งานระบบ Dashboard ผ่าน Google Sheets ของคุณ:

### ขั้นตอนที่ 1: เปิดพื้นที่เขียนโปรแกรมบน Google Sheets
1. เปิดเอกสาร Google Sheets ที่เก็บข้อมูลการสมัครของคุณ
2. ที่เมนูด้านบน เลือก **ส่วนขยาย (Extensions)** > **Apps Script**
3. ระบบจะเปิดหน้าต่างแก้ไขโค้ดของ Google Apps Script ขึ้นมา

### ขั้นตอนที่ 2: ติดตั้งโค้ดหลังบ้าน (`Code.gs`)
1. ในหน้า Apps Script Editor จะมีไฟล์ชื่อ `รหัส.gs` หรือ `Code.gs` อยู่แล้วเป็นค่าเริ่มต้น
2. ให้ลบโค้ดเริ่มต้นทั้งหมดออก
3. คัดลอกโค้ดทั้งหมดจากไฟล์ [Code.gs](file:///d:/ฟีฟาย/Code.gs) ในเครื่องของคุณไปวางในหน้าต่างแก้ไขโค้ด

### ขั้นตอนที่ 3: ติดตั้งโค้ดหน้าจอแสดงผล (`Index.html`)
1. ในแถบเมนูด้านซ้ายของ Apps Script Editor ให้คลิกเครื่องหมายบวก **`+`** ข้างคำว่า "ไฟล์"
2. เลือกประเภทไฟล์เป็น **HTML**
3. ตั้งชื่อไฟล์ว่า **`Index`** (ไม่ต้องพิมพ์ `.html` ระบบจะใส่ให้เอง)
4. ลบโค้ดเริ่มต้นทั้งหมดในไฟล์ `Index.html` ออก
5. คัดลอกโค้ดทั้งหมดจากไฟล์ [Index.html](file:///d:/ฟีฟาย/Index.html) ในเครื่องของคุณไปวางแทนที่

### ขั้นตอนที่ 4: บันทึกและทดลองใช้งาน
1. คลิกปุ่ม **บันทึกโครงการ (Save Project)** รูปแผ่นดิสก์ด้านบน หรือกดปุ่ม `Ctrl + S`
2. ทดลองรันโปรเจกต์โดยเลือกฟังก์ชัน `doGet` แล้วคลิก **เรียกใช้งาน (Run)** เพื่อให้ระบบบันทึกค่าระบบครั้งแรก (หากมีป็อปอัปขอสิทธิ์ ให้ดำเนินการให้สิทธิ์ตามขั้นตอนถัดไป)

### ขั้นตอนที่ 5: เผยแพร่เป็นเว็บแอปพลิเคชัน (Deploy)
1. คลิกปุ่ม **การทำงานร่วมกัน (Deploy)** สีน้ำเงินที่มุมขวาบน > เลือก **การใช้งานใหม่ (New deployment)**
2. คลิกรูปฟันเฟืองข้างคำว่า "เลือกประเภท (Select type)" > เลือก **แอปพลิเคชันเว็บ (Web app)**
3. ตั้งค่าการตั้งค่าดังนี้:
   * **คำอธิบาย (Description):** `Free Fire Registration Dashboard`
   * **เรียกใช้งานในฐานะ (Execute as):** เลือก **ฉัน (Me - อีเมลของคุณ)**
   * **ผู้ที่มีสิทธิ์เข้าถึง (Who has access):** เลือก **ทุกคน (Anyone)**
4. คลิกปุ่ม **ใช้งาน (Deploy)**
5. ระบบจะขึ้นหน้าต่างขอสิทธิ์การเข้าถึงข้อมูล (Authorization):
   * คลิก **ให้สิทธิ์เข้าถึง (Authorize access)**
   * เลือกบัญชี Google ของคุณ
   * หน้าต่างจะแจ้งเตือนว่า "Google hasn't verified this app" ให้คลิกตัวหนังสือเล็กๆ ด้านล่างว่า **ขั้นสูง (Advanced)**
   * คลิก **ไปที่โครงการที่ไม่มีชื่อ (ไม่ปลอดภัย) / Go to ... (unsafe)**
   * คลิก **อนุญาต (Allow)**
6. เมื่อเสร็จสิ้น คุณจะได้รับ **URL แอปพลิเคชันเว็บ (Web app URL)**
7. คัดลอกลิงก์ดังกล่าวส่งให้ทีมงานหรือเปิดบนเบราว์เซอร์เพื่อใช้งานได้ทันที!

## ⚡ คุณสมบัติเด่นของระบบ
* **Real-time Sync**: เมื่อมีทีมสมัครใหม่หรือมีการแก้ไขตัวสะกดใน Google Sheets ข้อมูลจะอัปเดตบน Dashboard ทันทีที่ผู้ใช้โหลดหน้าเว็บหรือกดปุ่ม 🔄 (Refresh)
* **Gaming / Esports Theme**: การออกแบบสไตล์เนียนสวย ดูล้ำสมัย ดึงดูดสายตา ตกแต่งด้วย Glassmorphism (กรอบโปร่งแสงเบลอหลัง)
* **Analytics Statistics**: แสดงยอดรวมจำนวนทีม, จำนวนผู้สมัครทั้งหมด, และสัดส่วนผู้สมัครจำแนกตามระดับชั้น (ม.1 - ม.6) แบบกราฟเปรียบเทียบในตัว
* **Multi-View Modes**: สลับหน้าจอการแสดงผลได้ 2 แบบ:
  * **แบบทีม**: แสดงเป็นการ์ดระบุชื่อทีม สมาชิกในทีมแบบย่อ สามารถคลิกเพื่อดูรายละเอียดเชิงลึกของลูกทีมทุกคนรวมถึงรหัสนักเรียนและเบอร์โทรศัพท์ได้
  * **แบบรายบุคคล**: แสดงรายชื่อผู้เข้าแข่งขันทั้งหมดในรูปแบบตาราง เหมาะสำหรับการคัดลอกรายชื่อไปจัดการในระบบอื่น หรือส่งข้อมูลต่อ
* **Advanced Filters**: ค้นหาได้อิสระจากชื่อทีม, ชื่อผู้สมัคร, รหัสนักเรียน หรือแม้กระทั่งเบอร์โทรศัพท์ และสามารถกรองเฉพาะระดับชั้นที่ต้องการได้ทันที
* **Offline Fallback**: หน้าเว็บทำงานได้เสถียร มีระบบข้อมูลสำรองติดตัวในกรณีที่เกิดปัญหาเน็ตหลุด จะมีไฟสถานะสีเหลืองแจ้งเตือน 🟡 และเปลี่ยนมาแสดงข้อมูลแคชที่ดึงมาล่าสุดโดยอัตโนมัติ
