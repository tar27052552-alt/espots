⚙️ ตั้งค่าระบบ

กรอก Firebase Config และชื่อทีมก่อนเริ่มการแข่งขัน

📋 วิธีสร้าง Firebase Project (ฟรี)
  1. ไปที่ console.firebase.google.com → คลิก "Add project"
  2. ตั้งชื่อ Project เช่น ff-esports-score → Next → Disable Analytics → Create
  3. เมื่อสร้างเสร็จ: คลิกไอคอน Web (</>) → ลงทะเบียน App → Copy ค่า firebaseConfig
  4. ไปที่เมนู Realtime Database → "Create database" → เลือก Asia-Southeast1 → Start in test mode
  5. กลับมากรอก API Key ด้านล่าง แล้วกด "บันทึกและทดสอบ"
